# Скрипт проверки орфографии и пунктуации через ChatGPT

Небольшой CLI-скрипт на Python, который отправляет текст в OpenAI API и возвращает исправленный вариант.

## Требования

- Python 3.9+
- Библиотека `requests`

Установить зависимости:

```bash
pip install requests
```

## Настройка

Экспортируйте API-ключ:

```bash
export OPENAI_API_KEY="ваш_ключ"
```

(Опционально) Можно задать модель:

```bash
export OPENAI_MODEL="gpt-4.1-mini"
```

## Где находится скрипт

Файл скрипта лежит в корне репозитория: `spellcheck.py`.

## Использование

Проверка текста аргументом:

```bash
python3 spellcheck.py --text "Это текст без запятых и с ошибкамы"
```

Также можно сделать файл исполняемым и запускать напрямую:

```bash
chmod +x spellcheck.py
./spellcheck.py --text "Это текст без запятых и с ошибкамы"
```

Проверка текста из stdin:

```bash
echo "это пример текста" | python3 spellcheck.py
```

## Примечания

- Скрипт возвращает только исправленный текст без пояснений.
- Температуру можно изменить флагом `--temperature` (по умолчанию 0.0).
